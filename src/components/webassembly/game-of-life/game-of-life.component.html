Conway's Game of Life using Rust/WebAssembly

The engine for setting up the universe for this implementation of Conway's Game of Life is written in Rust and compiled to WebAssembly. Thanks to this book for the Rust/WASM tutorial. The JS/HTML for this project is nearly all my own.

Instructions

See Conway's Game of Life for more information on the rules of the game.

  • Click the Randomize button to randomize the state of each cell in the universe
  • Click the Clear button to reset the universe to every cell being dead
  • Click on an individual cell to toggle its state
